This is a high-performance, brutalist-editorial engine designed to showcase projects, achievements, and literary experiments. It is not a template. Every element serves an intentional brand narrative, focusing on bold typography and raw technical execution.

CORE PHILOSOPHY
The project avoids the typical startup aesthetic in favor of a distinctive, art-driven identity. It relies on a custom-built CMS engine powered by Supabase, giving the builder full control over every pixel and data point.

TECHNICAL STACK
Frontend: React 18, Vite, TypeScript
Styling: Tailwind CSS v4, Radix UI, Framer Motion
Motion: GSAP and custom entrance choreography
Backend: Supabase (Auth, DB, Storage)
Routing: React Router DOM v6
State: TanStack Query

KEY ENGINES
1. Admin Dashboard: Full CMS for projects, social links, and wins.
2. F1 Pre-loader: High-velocity entrance with synchronized motion.
3. Project Gallery: An image-focused masonry grid with immersive details.
4. Wins Timeline: Dynamically sorted victory log for hackathons.

LOCAL SETUP
1. Clone the repository
2. Create a .env file with VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY
3. Run npm install
4. Run npm run dev

PRODUCTION
Run npm run build to generate the optimized bundle.

NOTES
This project uses the latest Tailwind CSS v4 alpha. Ensure your environment supports modern CSS-first configuration and Lightning CSS.
